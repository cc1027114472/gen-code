# OOXML.md

本文档用于承载 OOXML 相关的专项参考信息，例如 Word、Excel、PowerPoint 文档结构、打包约定和常见修改边界。

## 适合收录的内容

- OOXML 包结构说明。
- 常见部件路径与关系文件。
- 修改文档时的安全边界。
- 调试损坏文档时的排查顺序。

## 使用建议

- 主 `SKILL.md` 只负责提示“遇到 OOXML 细节时读取本文件”。
- 本文件聚焦结构真值，不重复主工作流。
- 对 XML 标签、路径和 schema 名称保持原文，叙述说明使用中文。

## 目标

让 Codex 在需要处理 OOXML 专项问题时，能快速拿到结构化参考，而不是在主文档中漫无目的搜索。

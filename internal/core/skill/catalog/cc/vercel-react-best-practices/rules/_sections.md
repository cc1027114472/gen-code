# 章节

该文件定义了所有章节、它们的顺序、影响级别和描述。
章节 ID（括号中）是用于对规则分组的文件名前缀。

---

## 1. 消除瀑布（async）

**影响：** 关键
**描述：** 瀑布是第一大性能杀手。每个顺序等待都会增加完整的网络延迟。消除它们会产生最大的收益。

## 2. 包体积优化（bundle）

**影响：** 关键
**描述：** 减少初始包大小可以缩短交互和最大内容绘制的时间。

## 3. 服务器端性能（server）

**影响：** 高
**描述：** 优化服务器端渲染和数据获取消除服务器端瀑布并减少响应时间。

## 4. 客户端数据获取（client）

**影响：** 中高
**描述：** 自动重复数据删除和高效的数据获取模式减少了冗余的网络请求。

## 5. 重新渲染优化（rerender）

**影响：** 中
**描述：** 减少不必要的重新渲染可以最大程度地减少浪费的计算并提高 UI 响应能力。

## 6. 渲染性能（rendering）

**影响：** 中
**描述：** 优化渲染过程减少了浏览器需要做的工作。

## 7. JavaScript 性能（js）

**影响：** 低-中
**描述：** 热路径的微观优化可以带来有意义的改进。

## 8. 高级模式（advanced）

**影响：** 低
**描述：** 针对需要谨慎实施的特定场景提供高级模式。
